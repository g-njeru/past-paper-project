# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it
privately using [GitHub's Security Advisory](https://github.com/g-njeru/past-paper-project/security/advisories/new)
feature. Please do **not** report security vulnerabilities through public
GitHub issues.

You should receive a response within 48 hours. If you don't, please follow up
to ensure your report was received.

## Scope

This project is a study tool for CPA candidates. It stores no sensitive user
data beyond phone numbers and display names. The Supabase backend is secured
via Row Level Security. Authentication is handled by Supabase Auth.
